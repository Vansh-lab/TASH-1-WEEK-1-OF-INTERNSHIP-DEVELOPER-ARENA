2-Understanding Programming Concepts?

Program: A set of instructions that a computer follows to perform a task. 
Syntax: Set of rules defning how to write valid statements. 
Interpreter: A program that executes Python code line-by-line, giving instant feedback and error checks. Python promotes modularity (break code into reusable "modules" and "functions"). Variables              and Data Types 
Variable: A variable is a named memory location that can hold data of different types and whose value can be changed during the execution of a program. 
Python variables are dynamically typed, i.e., it's not necessary to specify their type as it is determined at runtime.

Understanding Errors and Reading Them: 
Syntax Error: Incorrectly written code according to the rules of the programming language. Runtime Error: The code is executed but an error is encountered during the execution (for example dividing by zero). 
Traceback: Python displays the error type, the line number, and the error message which gives an indication of where the problem is. 
Understanding errors: Errors should be read from the bottom upwards. The last line (error type) indicates the most relevant information, then lines above show the chain of calls leading to the error.
